Clavis Password Manager

ローカル完結型パスワード管理 v0.0.3

ID・パスワード・URL・メモを AES-256-GCM でローカルに暗号化保存するパスワードマネージャ。 ID Manager 風のグループ階層 UI、CSV / XML / JSON のインポート・エクスポート、強力なパスワード生成器を備え、 クラウドにも外部にもデータを送りません。

最新版ダウンロード マニュアル

Windows 11 (64-bit) ・無料・フリーソフト

主な機能

AES-256-GCM 暗号化

金庫ファイルはマスターパスワードから PBKDF2-310000 で導出された鍵で AES-256-GCM 暗号化。

グループ階層管理

ID Manager 風のツリービュー。ドラッグ&ドロップで並べ替え・移動、グループのマージにも対応。

CSV / XML / JSON インポート

Shift-JIS / UTF-8 双方の自動判定で、ID Manager や他社マネージャからの移行も可能。

パスワード生成器

英大小・数字・記号と長さを指定可、暗号論的乱数 (crypto.randomBytes) で生成。

30 秒で消えるクリップボード

コピーしたパスワードは 30 秒後に自動でクリップボードから削除。

即時検索

名前・ID・URL・メモの横断検索。文字を打つたびにリストを絞り込み。

動作環境

  • Windows 11 (64-bit)
  • 必要 RAM 約 2GB 以上を推奨
  • 金庫ファイルは OS の userData フォルダ配下 (%APPDATA%\Clavis Password Manager\vault.enc) に保存
  • インターネット接続は不要 (完全オフラインで動作)

インストール

  1. 最新版リリースページから Clavis Password Manager Setup x.y.z.exe をダウンロード
  2. ダウンロードしたインストーラを実行
  3. スタートメニューから「Clavis Password Manager」を起動
  4. 初回起動時にマスターパスワードを設定 (忘れると復号不能のため厳重に管理)
