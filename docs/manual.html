マニュアル

Clavis Password Manager (passwordmgr.js) v0.0.3 の操作マニュアルです。ID・パスワード・URL をローカルに暗号化保存する Electron アプリです。

インストールと起動

  1. 最新版リリースから Clavis Password Manager Setup x.y.z.exe をダウンロード
  2. インストーラを実行 (インストール先・スタートメニュー登録・デスクトップショートカットを選択可能)
  3. スタートメニューまたはデスクトップから「Clavis Password Manager」を起動

初回起動時はウェルカムカルーセルが表示され、レイアウト・主要機能・ショートカットの紹介に続いてマスターパスワードの登録を行います。後からも右上の ボタンでチュートリアルを再表示できます。

マスターパスワードを忘れると保存データを復元できません。必ず安全な場所にメモしてください。

基本操作

ロック解除後は 3 パネルレイアウトで操作します。

  • 左パネル — グループツリー。フォルダ階層でエントリを分類
  • 右上パネル — 選択中のグループに含まれるエントリ一覧 (タイトル・ID・URL・更新日時)
  • 右下パネル — 選択エントリの詳細編集フォーム

パネルの境界 (リサイザー) はドラッグして幅・高さを調整できます。

エントリの追加・編集・削除

  • 新規エントリ — ツールバーの New ボタン (Ctrl+N)
  • 新規グループ — ボタン。グループ名を入力
  • 編集 — エントリをクリックして右下フォームで Title / ID / Password / URL / Notes を編集 → Save
  • 削除 — エントリを選択して または Delete キー
  • 並べ替え・移動 — グループツリーやエントリ一覧で右クリックメニューから操作

コピー操作

各フィールド右の ボタンで ID・パスワード・URL をワンクリックでクリップボードへコピーできます。 ボタンで URL を既定ブラウザで開きます。

主な機能

パスワード生成

ツールバーの ボタン、または詳細パネル右の ボタンでパスワード生成モーダルを開きます。

  • 長さ — 4 〜 64 文字 (スライダー)
  • 文字種 — 大文字 / 小文字 / 数字 / 記号 を個別に ON/OFF
  • Generate で再生成、Use で現在編集中のエントリに反映、 でコピー

パスワード表示・一括表示

詳細パネルの ボタンで個別エントリのパスワードを表示/非表示。ツールバーの でリスト内すべてのパスワードを一括表示/非表示できます。

検索

ツールバー右の検索ボックスにキーワードを入力するとタイトル・ID・URL・Notes を横断検索します (Ctrl+F でフォーカス)。 でクリア。

インポート

  • XML — ID Manager 形式の XML ファイルをインポート
  • CSV — 一般的な CSV (Title, Username, Password, URL, Notes など)
  • JSON — Clavis 形式の JSON

ファイル選択後、インポート設定モーダルで取り込み先フォルダを指定できます:

  • ファイル名で新規フォルダ作成 (デフォルト)
  • 既存フォルダに追加
  • 任意の名前で新規フォルダ作成
  • "Imported" フォルダに追加

エクスポート

ツールバーの CSV / JSON ボタンで全データを書き出します。バックアップや他ツールへの移行に利用してください。

エクスポートファイルは平文です。取り扱いには十分注意し、不要になったら安全に削除してください。

ロック

ツールバーの ボタンで即座にロック画面に戻ります。再開時にはマスターパスワードの入力が必要です。

ダーク／ライトテーマ

ツールバー右の ボタンでテーマを切替。設定は次回起動時にも保持されます。

キーボードショートカット

Ctrl+N新しいエントリを作成
Ctrl+S編集中のエントリを保存
Ctrl+F検索ボックスにフォーカス
Delete選択したエントリを削除
Escコンテキストメニュー・モーダルを閉じる

設定

  • マスターパスワードの変更 — 初回登録後の変更は、現在のデータをエクスポート → ロック → 新規 Vault 作成 → 再インポートの手順で行えます
  • テーマ — ツールバーの ボタンで切替。選択はローカルに保存
  • パネル幅・高さ — リサイザーで調整した値は次回起動時にも保持されます
  • アプリ情報 — ボタンで バージョン・コミット・Electron バージョン・作者を表示

セキュリティ

Clavis はクラウドを利用せず、すべてのデータをローカルに暗号化して保存します。

  • 暗号化アルゴリズム — AES-256-GCM (認証付き暗号)
  • 鍵導出 — PBKDF2-HMAC-SHA256 / 反復回数 310,000 / ソルト長 32 バイト (エントリ単位で乱数生成)
  • IV (初期化ベクトル) — 12 バイトの乱数を毎回生成
  • 認証タグ — GCM の authTag により改ざんを検知。検証失敗時は復号エラーとなり一切のデータを返しません
  • マスターパスワード — 平文では保存されず、復号成功の可否でのみ検証されます
  • 保存場所 — Electron の userData ディレクトリ内 (Windows: %APPDATA%\\Clavis Password Manager\\)
  • 外部通信なし — クラウド同期や解析サービスへの送信は行いません
マスターパスワードの紛失は復旧不可能です。定期的にエクスポート (CSV/JSON) でバックアップを取ることを推奨します。バックアップファイルは平文のため、暗号化されたディスクや安全な場所に保管してください。
